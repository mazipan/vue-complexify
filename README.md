## How to use

#### Import plugins
```javascript
import VueComplexify from 'vue-complexify'
Vue.use(VueComplexify, {})
```

#### Using in component
```javascript
this.$evalPasswordComplexity(passwordText, this.callback)
```

#### Callback parameter
```javascript
function callback(err, valid, complexity) {
  console.log(err, valid, complexity)
}
```
